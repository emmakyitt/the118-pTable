# THE118™ 元素周期表 🏌🏻‍♂️• PTE
## 简介 (Introduction)
**the118-pTable** 是一款基于 *React、TypeScript 与 Vite* 技术生态构建的**现代 Web 应用**，致力于在浏览器端呈现**高沉浸感的交互式**化学元素周期表可视化方案。项目内置多种风格化的**三维布局视图**，并以**细腻的动画过渡**确保视图间无缝切换。在实现层面，其核心采用**空间映射算法**根据参数化坐标生成视图模型。通过精确的数学算法解算元素空间坐标，进而依托坐标向量值结合*CSS3 Matrix3D*技术，驱动三维布局模型的动态变换效果，实现页面元素动效的高性能渲染。

## 核心特性 (Features)
项目提供了五种布局视图，通过页面底部的按钮进行切换，每种布局都为元素卡片设定了不同的位置和旋转角度。
- **表格 (TABLE)**：模拟传统的二维元素周期表样式，按照其在周期表中的**族**和**周期** 进行排列
- **球体 (SPHERE)**：所有元素分布在一个虚拟球体的表面上，形成一个壮观的**3D球体**结构
- **螺旋 (HELIX)**：元素沿着一条三维螺旋线排列，形似一个**旋转的DNA链**或弹簧
- **网格 (GRID)**：元素在一个三维空间中排列**成规则**的网格
- **柱形 (cylinder)**: 螺旋形态的演变，元素在排列上更富有层次感
